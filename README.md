# 🤖 AI Tech Stack Recommender

An AI-powered recommendation system that suggests the most relevant technology stacks and career paths based on user skills using **Content-Based Filtering**, **TF-IDF Vectorization**, and **Cosine Similarity**.

---

## 🚀 Project Overview

This project was developed as part of the **DecodeLabs Artificial Intelligence Internship Program**.

The recommendation engine analyzes user-provided skills and intelligently matches them with predefined job roles and technology stacks to generate personalized recommendations.

---

## ✨ Features

- User Skill Input
- Content-Based Recommendation System
- TF-IDF Feature Extraction
- Cosine Similarity Scoring
- Top-N Recommendations
- Career Path Suggestions
- Fast and Lightweight Implementation

## 📊 Working Process

### Step 1: User Input

Example:

```text
python, machine-learning, sql
```

### Step 2: TF-IDF Vectorization

Converts skills into numerical vectors.

### Step 3: Cosine Similarity

Measures similarity between user skills and job profiles.

### Step 4: Recommendation

Returns the Top 3 most relevant career paths.

Example Output:

```text
Top Recommendations

1. Data Scientist
2. AI Engineer
3. Data Analyst
```
