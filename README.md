# AI Tech Stack Recommender

## Overview

An AI-powered recommendation system that suggests
career paths based on user skills using Content-Based
Filtering and Cosine Similarity.

## Features

- User Skill Input
- TF-IDF Vectorization
- Cosine Similarity Matching
- Top 3 Recommendations
- Content-Based Recommendation Logic

## Technologies

- Python
- Pandas
- Scikit-Learn
